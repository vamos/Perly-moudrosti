IMS project - Vole crisis
=========================

The program 'ca' is simulating the vole population in area of 1 Ha.
The time in simulation starts at first day of march. The time step
between iterations is 25 days. It is implemented as a cellular automaton. 

For compilation use command 'make' in root directory of this project.

Usage:

	./ca 		<- runs in default configuration 
			   number of iteration is set to 14 which
			   covers 1 year

	./ca -h 	<- prints help and all of possible options
